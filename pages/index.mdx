# POTATO bot

POTATO is a discord bot that has jokes, Minecraft intergrations, and basic moderation features.

It's written in [typescript](https://typescriptlang.org) and is powered by [discord.js](https://discord.js.org/) and [discord-akairo](https://discord-akairo.github.io/).

It's also open source [on github](https://github.com/edazpotato/potato) if you want to see how something is done or make a contribution.
