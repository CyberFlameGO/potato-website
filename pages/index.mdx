# POTATO bot

POTATO is a discord bot that has loads of funny jokes.

It's written in [TypeScript](https://typescriptlang.org) and is powered by [Discord.js](https://discord.js.org/).

It's also open source [on GitHub](https://github.com/edazpotato/POTATO) if you want to see how something is done or make a contribution.
