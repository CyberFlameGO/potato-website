# Getting started with POTATO

## Adding POTATO to a server

To add POTATO to a Discord server, visit this link and follow the on screen guides: [https://potato.edaz.codes/links/add](/add)

### Commands

POTATO's commands are only avalible through Discord's slash-commands system. This means that they won't stop working in the future.
